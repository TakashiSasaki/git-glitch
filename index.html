Gist Writer for ChatGPT

A tool to simplify sharing and managing code snippets

Overview

Gist Writer is a custom GPT that runs on the OpenAI ChatGPT platform. It allows you to create Gists directly from within ChatGPT and load Gist files into your conversation. This helps you organize and share information without interrupting the flow of conversation.

Gist Writer uses the GitHub REST API to interact with GitHub Gists. It supports the following operations:

  • Retrieve a list of Gists created by the user account
  • Create a new Gist (defaulting to private settings)
  • Retrieve the content of a specific Gist by its ID
  • Modify the content of an existing Gist by specifying its ID
  • Delete a Gist based on its ID

When a new Gist is created, the generated Gist ID and URL are displayed for easy access. If a Gist URL is provided, Gist Writer extracts the Gist ID from the URL and performs the requested action. If no filename or content is specified, default settings are used.
