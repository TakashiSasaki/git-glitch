flatte: A Python Module for Flattening and Unflattening JSON

The flatte module provides Python functions to flatten nested JSON-like structures into a list of tuples and to unflatten them back to their original structure. This is particularly useful for easily manipulating or analyzing deeply nested JSON data.

Visit the following links for source code and documentation:

  • Documentation
  • flatten.py
  • unflatten.py

Using flatte via httpimport

You can easily use the flatte module via httpimport. This allows you to import Python modules from a web server directly. Below are some examples demonstrating how to use the flatten and unflatten functions. 

    
from httpimport import remote_repo
with remote_repo("https://flatte.glitch.me/"):
  import flatte

Example: Using flatten


print(flatte.flatten("abc"))
>>> [('', 'abc')]

Example: Using unflatten


>>> print(flatte.unflatten([('', 'abc')]))    
abc
